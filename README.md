# CODSOFT Machine Learning Internship Projects

This repository contains my submissions for the **CODSOFT Machine Learning Internship**.  
The internship consists of 5 practical tasks aimed at building machine learning models and AI applications. Completion of at least 3 tasks is required to successfully complete the internship.

---

## Internship Objective
The purpose of this internship is to **learn and grow** by implementing real-world machine learning tasks.  
The projects cover areas such as classification, prediction, fraud detection, spam detection, and AI-based text generation.

---

## Tasks Overview

| Task No | Task Name | Description |
|---------|-----------|-------------|
| 1 | [Movie Genre Classification](./Task1_Movie_Genre_Classification/README.md) | Predict movie genres based on plot summaries using ML techniques like TF-IDF and Naive Bayes, Logistic Regression, or SVM. |
| 2 | [Credit Card Fraud Detection](./Task2_Credit_Card_Fraud/README.md) | Detect fraudulent credit card transactions using ML models such as Logistic Regression, Decision Trees, or Random Forests. |
| 3 | [Customer Churn Prediction](./Task3_Customer_Churn/README.md) | Predict which customers are likely to leave a subscription-based service using models like Random Forest, Logistic Regression, or Gradient Boosting. |
| 4 | [Spam SMS Detection](./Task4_Spam_SMS_Detection/README.md) | Classify SMS messages as spam or legitimate using text processing techniques and classifiers such as Naive Bayes or SVM. |
| 5 | [Handwritten Text Generation](./Task5_Handwritten_Text_Generation/README.md) | Generate handwritten-like text using character-level RNN/LSTM trained on handwritten text datasets. |
